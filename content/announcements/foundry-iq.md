## 概要

Foundry IQ は Microsoft Foundry のナレッジプレーンとして、エージェントにエンタープライズデータとウェブ情報の統合コンテキストを提供する。Build 2026 で Public Preview として発表され、カスタム RAG パイプラインの構築を不要にすることを目指す。

## 主な発表

- **Foundry IQ**: 統合ナレッジプレーン（**Public Preview**）
- **サーバーレス検索**: インフラ管理不要の検索基盤
- **ナレッジベース**: 構造化されたナレッジソースの管理
- **Web IQ 統合**: ウェブからのリアルタイム情報取得
- **エージェンティック検索改善**: エージェント向け検索の品質向上

## 詳細

### Microsoft IQ ファミリー

Foundry IQ は Microsoft IQ の一部として、他の IQ レイヤーと統合される:

- **Work IQ**: Microsoft 365 のワークプレースインテリジェンス
- **Fabric IQ**: Microsoft Fabric のビジネスモデルインテリジェンス
- **Foundry IQ**: ナレッジ検索とグラウンディング
- **Web IQ**: ウェブからのリアルタイムグラウンディング

## 応用シナリオ

- エンタープライズドキュメントとウェブ情報を統合したリサーチエージェント
- カスタム RAG パイプラインの代替としての Foundry IQ 利用
- 複数のナレッジソースを統合したカスタマーサポートエージェント
