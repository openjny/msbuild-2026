## 概要

Azure Logic Apps は Build 2026 でエージェント統合とマルチエージェンティック自動化を中心に機能強化を発表した。MCP サーバーとしての Logic Apps 公開や、A2A（Agent-to-Agent）標準対応の会話型エージェントが主要なテーマである。

## 主な発表

- **Azure Logic Apps Build 2026 更新**: エージェントループ、MCP サーバー統合の強化
- **MCP サーバーとしての Logic Apps**: AI エージェントから既存ワークフローを再利用
- **マルチエージェンティック対応**: A2A 標準準拠の会話型エージェント

## 詳細

### MCP サーバー統合

Logic Apps（Standard）を MCP サーバーとして公開し、AI エージェントが既存の1400以上のコネクタを活用したエンタープライズワークフローを安全に呼び出せる。

## 応用シナリオ

- 既存の Logic Apps ワークフローを AI エージェントから MCP 経由で再利用
- 複数のエージェントが A2A 標準で連携するビジネスプロセス自動化
- 1400以上のコネクタを活用したエンタープライズ統合のエージェント化
