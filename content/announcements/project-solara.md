## 概要

Project Solara は、エージェントを第一級のインターフェースとして設計されたデバイスプラットフォームである。従来のアプリ中心のデバイスとは異なり、AI エージェントがユーザーとのインタラクションの主要な手段となる。

Microsoft Device Ecosystem Platform（MDEP）は Android をベースとしたエンタープライズ向け OS で、チップからクラウドまでの統合スタックとして設計されている。

## 主な発表

- **Project Solara プラットフォーム**: エージェントファーストのデバイスプラットフォーム
- **2つの参照デザイン**: デスクデバイスとウェアラブルバッジのコンセプトデバイス
- **プライベートパイロットプログラム**: AccuWeather、Best Buy、CVS Health、Levi's、Target 等との試験導入

## 詳細

### エージェント開発

以下の既存ツールで Project Solara デバイス向けエージェントを構築可能:

- Microsoft 365 Copilot の宣言型エージェントまたはカスタムエンジンエージェント
- Copilot Studio
- Microsoft 365 Agents SDK
- Microsoft Agent Framework

### 応用領域

- GitHub Copilot: 音声などの新しいモダリティで開発者の生産性向上
- Dragon Copilot: 医療従事者向けのケアフローでのエージェント体験
- 小売・サービス業での業務用エージェントデバイス
