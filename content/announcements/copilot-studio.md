## 概要

Microsoft Copilot Studio は Build 2026 でエージェントプラットフォームとしての機能を大幅に拡張した。コンピューター操作エージェント、ワークフロー、相互運用性、リアルタイム音声にわたる更新により、エンタープライズ向けエージェントの構築・運用を加速する。

## 主な発表

- **コンピューター操作エージェントの強化**: UI 操作によるエンドツーエンドの業務自動化
- **新ワークフロー体験**: エージェントとワークフローの統合
- **リアルタイム音声エージェント**: 音声ベースのエージェント体験
- **新オーケストレーションレイヤー**: 評価パフォーマンス約 20% 向上、トークン消費 50% 削減
- **Azure Databricks Genie 連携**: MCP 経由でのデータ分析エージェント構築
- **Agent Builder → Copilot Studio 移行**: Agent Builder で作成したエージェントを Copilot Studio に移行して高度な機能を利用

## 詳細

### コンピューター操作エージェント

受信メールの解釈、ビジネスルールへの検証、UI を通じたシステム操作、例外時のワークフローエスカレーションを自動化。Graebel 社の事例では、サービスオーダー処理のエンドツーエンド自動化を実現。

### 新オーケストレーションレイヤー

アップグレードされた AI スタックにより、マルチステップ業務プロセスの実行精度と効率が向上。Early Release 環境で自動適用中。

### Agent Builder との関係

Agent Builder で個人やチーム向けに素早く作成したエージェントを、Copilot Studio に移行してトピック、オートメーション、公開チャネルの拡張、DLP・ALM などのガバナンス機能を追加可能。
