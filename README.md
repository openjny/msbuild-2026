# Microsoft Build 2026

Microsoft Build 2026 で発表された情報を集約したハブサイトです。日本人向けの Web ページ、AI エージェント向けの llms.txt / .well-known、VS Code Agent Plugins 向けの skills を、同じコンテンツソースから生成して配信します。

## 🌐 WEB サイト

<https://openjny.github.io/msbuild-2026/>


## 🤖 AI エージェント向け WEB サイト
 <https://openjny.github.io/msbuild-2026/llms.txt>

使用例：

```txt
私はネットワークエンジニアです。
今回の Build の発表やセッションから読み解けるインサイトを教えて。

情報元: https://openjny.github.io/msbuild-2026/llms.txt
```

## 🤖 Agent Skill

VS Code / GitHub Copilot:

```
/plugin install openjny/msbuild-2026
```
