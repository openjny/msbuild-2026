# Azure App Service

Azure App Service の Build 2026 更新として、エージェンティックワークロード対応の強化、Easy Auth によるエージェント認証の簡素化、パフォーマンス改善を含む複数の機能更新を発表。AI エージェントのホスティング基盤としての App Service の位置づけを強化する。

## 概要

Azure App Service は Build 2026 で複数の機能更新を発表した。クラウドネイティブアプリケーションとエージェンティックワークロード向けの基盤強化が中心テーマである。
