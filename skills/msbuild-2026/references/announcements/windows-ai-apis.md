# Windows AI APIs

Windows AI APIs のハードウェア対応を Copilot+ PC を超えて拡大。Speech Recognition API を Preview で提供、新 SLM Aion 1.0 Instruct を Preview で公開、14B パラメータの推論・ツール呼び出しモデル Aion 1.0 Plan を近日提供予定。ローカル AI をより幅広い Windows デバイスで利用可能にする。

## 概要

Windows AI APIs は Build 2026 で Copilot+ PC を超えてハードウェア対応を拡大した。新しいローカル SLM と API により、より幅広い Windows デバイス上でローカル AI を実行可能にする。

## 主な発表

- **Speech Recognition API**: 音声認識のローカル実行（**Preview**）
- **Aion 1.0 Instruct**: 新しい Small Language Model（**Preview**）
- **Aion 1.0 Plan**: 14B パラメータの推論・ツール呼び出しモデル（**近日**）

## 詳細

### Aion 1.0 Instruct

Windows 向けに最適化された新しい SLM。ローカルデバイス上での推論に対応し、クラウド接続なしで AI 機能を提供。

### Aion 1.0 Plan

14B パラメータの推論モデル。ツール呼び出し機能を備え、エージェントのローカル推論・計画立案に対応。

### Speech Recognition API

音声認識をローカルで実行する API。クラウドに依存せず、プライバシーを保ちながら音声入力を処理。
