# Windows 365

Windows 365 の Build 2026 更新として、エージェントランタイム、32vCPU Cloud PC、GPU Select プラン、Autopilot device preparation、Azure Compute Gallery サポート GA を発表。開発者とエージェントのためのクラウド PC プラットフォームを強化。

## 概要

Windows 365 は Build 2026 で「開発者とエージェントのためのクラウド PC」として複数の強化を発表した。コンピュートオプションの拡大、エージェントランタイムとしての活用、開発環境の自動プロビジョニングが主要なテーマである。

## 主な発表

- **32vCPU Cloud PC**: Windows 365 Enterprise / Flex で利用可能
- **GPU Select プラン**: GPU 対応 Cloud PC ポートフォリオの拡大
- **Autopilot device preparation**: アプリ・スクリプトの自動インストール
- **Azure Compute Gallery サポート**: カスタムイメージの管理（**GA**）
- **エージェントランタイム**: Computer-Using Agents の実行基盤としての Windows 365

## 詳細

### コンピュートオプション

32vCPU Cloud PC はソフトウェア開発、データモデリング、シミュレーション、AI/ML などの計算集約型ワークロードをサポート。

### 開発環境のカスタマイズ

今後のプレビューで、SDK、CLI、パッケージ、ビルドツール、リポジトリ、オンボーディングワークフローのカスタマイズが可能に。エンタープライズガードレール内で開発者が即座に生産的になれる環境を提供。

## 応用シナリオ

- Computer-Using Agent が Windows 365 上で Web アプリケーションを操作・テスト
- 開発チーム向けの標準化されたクラウド開発環境の自動プロビジョニング
- GPU 対応 Cloud PC での AI/ML 開発
