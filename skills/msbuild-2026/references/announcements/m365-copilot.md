# Microsoft 365 Copilot

Microsoft 365 Copilot の Build 2026 更新として、新デザイン（チャット中心のストリームライン UI）、Copilot Health（Preview）、Copilot Notebooks の強化（インフォグラフィック生成、Excel 連携）、Apple CarPlay 対応を発表。Autopilot カテゴリの導入で Copilot エコシステム全体が拡大。

## 概要

Microsoft 365 Copilot は Build 2026 でデザイン刷新と新機能を発表した。チャット中心のストリームラインされた UI、Copilot Health のプレビュー、Notebooks の拡張が主なアップデートである。Autopilot カテゴリ（Microsoft Scout）の導入により、Copilot エコシステム全体が拡大した。

## 主な発表

- **新デザイン**: チャット中心のストリームライン UI、ナビゲーション改善
- **Copilot Health**: 健康関連の AI アシスタント（**Preview**）
- **Copilot Notebooks 強化**: インフォグラフィック生成、Excel エージェント連携
- **Apple CarPlay 対応**: Microsoft 365 Copilot アプリの CarPlay サポート
- **Autopilot エコシステム**: Scout を皮切りに、Copilot アプリ内で Autopilot を構築・管理

## 詳細

### 新デザイン

チャットと応答レイアウトの簡素化、ナビゲーションの改善、作業整理の新しい方法を導入。ユーザーが集中を維持し、コンテキストを保ち、効率的に作業を前進させる。

### Copilot Notebooks

- インフォグラフィック生成: ノートブックの内容から視覚的サマリーを自動生成
- Excel エージェント: ノートブックからスプレッドシートを直接生成
