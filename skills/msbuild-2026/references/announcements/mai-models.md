# MAI Models

Microsoft AI が自社開発した7つの新モデルファミリーを発表。推論特化の MAI-Reasoning-1、コーディング特化の MAI-Code-1 / MAI-Code-1-Flash、画像生成の MAI-Image-1、音声の MAI-Voice-2、文字起こしの MAI-Transcribe-1.5 を含む。超知能研究所（Superintelligence Lab）の設立も発表し、Humanist Superintelligence を掲げた長期的な AI 能力向上を目指す。

## 概要

Microsoft AI（MAI）チームは Build 2026 で自社開発の7つの新モデルを発表した。コーディング、推論、画像生成、音声、文字起こしの各領域をカバーし、GitHub Copilot、Microsoft 365、Microsoft Foundry を通じて利用可能となる。

同時に超知能研究所（Superintelligence Lab）の設立を発表。Humanist Superintelligence — 人間に奉仕し、人間の監督に従い、人間の意図に沿う高度な AI システム — を長期目標として掲げた。

## 主な発表

- **MAI-Reasoning-1**: Microsoft 初の推論モデル（**GA**）
- **MAI-Code-1**: 推論効率に優れたコーディングモデル。GitHub Copilot と VS Code で利用可能（**GA**）
- **MAI-Code-1-Flash**: 小型コーディングモデル。サイズあたり最高クラスの品質（**GA ロールアウト中**）
- **MAI-Image-1**: 画像生成・画像間変換モデル。Arena AI リーダーボード2位（**GA**）
- **MAI-Voice-2 / MAI-Voice-2-Flash**: 15以上の追加言語と新しい音声オプション（**GA**）
- **MAI-Transcribe-1.5**: 43言語対応の文字起こしモデル。ストリーミング対応予定（**GA**）
- **超知能研究所**: 長期的 AI 能力研究組織の設立

## 詳細

### モデルの提供先

MAI モデルは Microsoft Foundry に加え、Fireworks AI、Baseten、Open Router でも利用可能。Fireworks AI は Foundry 上で **GA** となり、エンタープライズガバナンスと Azure データレジデンシーを維持しつつサードパーティモデルも利用できる。

### MAI-Image-1

PowerPoint で利用可能、OneDrive にロールアウト中。Foundry での提供も開始。市場をリードする品質/コスト比を実現。

### MAI-Code-1

GitHub 向けにチューニングされた推論効率の高いコーディングモデル。GitHub Copilot と VS Code で利用可能。

## 応用シナリオ

- GitHub Copilot のコード補完・エージェントモードでの MAI-Code-1 利用
- Foundry 上でのマルチモーダルエージェント構築（推論 + 画像 + 音声）
- PowerPoint・OneDrive での AI 画像生成による資料作成効率化
