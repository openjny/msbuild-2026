# GitHub Code Security

Build 2026 で GitHub のセキュリティ・プラットフォーム関連として、Azure Repos 向け Copilot Code Review（Technical Preview）、Code Review のチームカスタマイズ、Agent Apps（エージェント拡張機能）を発表。GitHub Copilot のセキュリティとプラットフォーム拡張性を強化。

## 概要

GitHub は Build 2026 で Copilot Code Review の拡張と Agent Apps による GitHub プラットフォームの拡張性強化を発表した。Code Review が Azure Repos にも対応し、チームごとのカスタマイズが可能に。Agent Apps でサードパーティが GitHub のエージェント機能を拡張できる。

## 主な発表

- **Copilot Code Review for Azure Repos**: Azure Repos 対応（**Technical Preview**）
- **Code Review カスタマイズ**: チーム固有のレビュー基準の設定
- **Agent Apps**: GitHub のエージェント機能をサードパーティが拡張

## 詳細

### Azure Repos 対応

GitHub Copilot のコードレビュー機能が Azure DevOps の Azure Repos でも利用可能に。GitHub リポジトリ以外の環境でもエージェント駆動のコードレビューを提供。

### Code Review カスタマイズ

チームのコーディング規約やベストプラクティスに基づいてレビュー基準をカスタマイズ可能。

### Agent Apps

サードパーティ開発者が GitHub のエージェント機能を拡張するためのアプリケーション。Copilot のツールやワークフローを外部から拡張できる。
